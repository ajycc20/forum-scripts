# forum-scripts

> 这里是三个脚本，主要用于定时发帖，定时回复，定时发豆
> 
> 事实上只是为了简化平时手动的操作才写的

* `checkin.js` 定时发帖(签到帖)
* `getbingurl.js` 定时回帖(我个人置顶的论坛必应壁纸贴)
* `givebonus.js` 定时发豆(对签到帖在每天固定时刻发送魔力豆)

> 其余详细代码介绍写在注释

## 运行

```bash
# Clone
git clone https://github.com/ajycc20/forum-scripts.git

# Install
npm install or yarn

# Setting
填写 cookie 

# Start
node index.js
```
